How to edit a student profile?

  1. Login into your account with the credentials that you received from the Uniwatcher admin team. If you are unable to log in, kindly contact the Uniwatcher team.

  2. Go to the student's profile by clicking on the student ID. You can see the ID in the dashboard or search via the Search Student menu.

  3. Once you make the necessary changes, click the Update button to save the changes.
