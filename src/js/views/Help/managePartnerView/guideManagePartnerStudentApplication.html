How to create an application for student?

  1. Login into your account with the credentials that you received from the Uniwatcher admin team. If you are unable to log in, kindly contact the Uniwatcher team.

  2. Go to the student's profile by clicking on the student ID. You can see the ID in the dashboard or search via the Search Student menu.

  3. Click on the Application tab at the top.

  4. Here you can see all the application-related enquiries.

  5. Fill out the required information and click on the Add button.

  6. Once an application is added, it will be listed like this. You can edit the application by clicking on the edit icon.
