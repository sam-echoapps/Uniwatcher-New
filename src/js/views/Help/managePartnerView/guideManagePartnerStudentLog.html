How to add log in student profile?

  1. Login into your account with the credentials that you received from the Uniwatcher admin team. If you are unable to log in, kindly contact the Uniwatcher team.

  2. Go to the student's profile by clicking on the student ID. You can see the ID in the dashboard or search via the Search Student menu.

  3. Scroll down to the Student Log section at the bottom. Click on the Add log button on the right side.

  4. Add in the required information. If you want to get a reminder alert via email for a particular matter, add the date. Click on the submit button.

  5. You can also edit the log as well by clicking on the edit button.

  6. You can mark reminders as done by clicking on the tick icon.
