How to search for a student in Uniwatcher CRM?

  1. Login into your account with the credentials that you received from the Uniwatcher admin team. If you are unable to log in, kindly contact the Uniwatcher team.

  2. Select the Search Student option from the left side menu.

  3. Type in the name you want to search. This will give you suggestions as you type in.

  4. Click on the student ID to view the profile of the student in a new tab.
