How to manage a partner profile?

  1. Login into the CRM.

  2. Select the Partners option from the left side menu.

  3. Select the name of the partner from the dropdown.

  4. Once you select the partner, click the Show button to see their profile.

  5. In the details tab, you can see all the basic details regarding the partner. Once you make any changes in the Details section, make sure to click on the Update button.

  6. In the Applications tab, you will be able to view the applications report by selecting the required filter.

  7. In the Final Choice tab, you will be able to view the final choice report by selecting the required filter.

  8. In the Contract Files tab, you can upload contract-related documents by clicking on the selection box. Once a file is selected, make sure to click on the Update button.

  9. In the Add Logs tab, you can add logs for the partner by clicking on the Add log button.

  10. Once you enter the log info click on the submit button to save it.

  11. In the Credentials tab, you will find the login credentials for the Partner. At first, it will be an auto-generated password.

  12. If you are okay with it, click on the Submit button. Once it is submitted, the password can be shared with the Partner by clicking on the Send button. Or you can add a new password and click on the Update button to save it, and then send the password.
